Каждый раз, когда вы посещаете веб-сайт, ваш браузер передаёт удивительное количество личных данных — ваш **публичный IP-адрес**, примерное физическое местоположение, операционную систему, модель GPU, разрешение экрана, уровень заряда батареи и даже локальные IP-адреса через утечки WebRTC. Рекламодатели и трекеры используют эти сигналы для создания **уникального цифрового отпечатка**, который может отслеживать вас по всему интернету, даже без файлов cookie. Большинство людей не подозревают, сколько данных они раскрывают, просто открывая вкладку браузера.

NetGuard делает невидимое видимым. Нажмите **Run Privacy Scan**, чтобы запустить комплексный аудит всего, что ваш браузер раскрывает прямо сейчас. Инструмент определяет ваш публичный IP и геолокацию, обнаруживает отпечаток оборудования (ОС, браузер, рендерер GPU через WebGL), проверяет **статус батареи** и скорость соединения, а также проводит **тест утечки WebRTC**, чтобы увидеть, раскрыты ли ваши приватные IP-адреса. Каждая точка данных отображается на панели в хакерском стиле с итоговым **Рейтингом Конфиденциальности** от 0 до 100. Всё работает в вашем браузере — никакие данные не сохраняются и не отправляются на наши серверы.
