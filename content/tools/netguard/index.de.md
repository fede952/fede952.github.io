Jedes Mal, wenn Sie eine Website besuchen, sendet Ihr Browser eine überraschende Menge an persönlichen Daten — Ihre **öffentliche IP-Adresse**, ungefähren physischen Standort, Betriebssystem, GPU-Modell, Bildschirmauflösung, Akkustand und sogar lokale Netzwerk-IPs durch WebRTC-Lecks. Werbetreibende und Tracker nutzen diese Signale, um einen **einzigartigen digitalen Fingerabdruck** zu erstellen, der Ihnen durchs Web folgen kann, auch ohne Cookies. Die meisten Menschen haben keine Ahnung, wie viele Daten sie allein durch das Öffnen eines Browser-Tabs preisgeben.

NetGuard macht das Unsichtbare sichtbar. Klicken Sie auf **Run Privacy Scan**, um ein umfassendes Audit von allem zu starten, was Ihr Browser gerade preisgibt. Das Tool ermittelt Ihre öffentliche IP und Geolokation, erkennt Ihren Hardware-Fingerabdruck (OS, Browser, GPU-Renderer via WebGL), prüft Ihren **Akkustatus** und die Verbindungsgeschwindigkeit und führt einen **WebRTC-Leak-Test** durch, um zu sehen, ob Ihre privaten IP-Adressen exponiert sind. Jeder Datenpunkt wird auf einem Hacker-Style-Dashboard mit einem abschließenden **Datenschutz-Score** von 0 bis 100 angezeigt. Alles läuft in Ihrem Browser — keine Daten werden gespeichert oder an unsere Server gesendet.
