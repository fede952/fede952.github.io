**PentaFramework** is a custom tool written in **Python** designed to streamline the execution of security assessments. It provides a modular structure that allows Red Teamers and Pentesters to automate repetitive tasks such as reconnaissance, scanning, and exploitation.

The main goal of this project is to create a unified interface for various security tools, ensuring consistency and speed during engagements.
