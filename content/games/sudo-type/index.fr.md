Menaces détectées. Le mainframe est attaqué. Votre seule arme ? **Votre clavier.**

Tapez les mots qui tombent avant qu'ils n'atteignent le mainframe. Enchaînez les combos pour des points bonus. Combien de temps pouvez-vous défendre le système ?

## Comment Jouer

- **Tapez** les mots qui tombent pour les détruire avant qu'ils ne touchent le mainframe.
- **Le ciblage** démarre automatiquement — tapez la première lettre d'une menace.
- **Les combos** multiplient votre score — enchaînez les éliminations sans rater.
- **La vitesse augmente** avec chaque niveau. Restez concentré.
- **Touchez** l'écran sur mobile pour ouvrir le clavier.
