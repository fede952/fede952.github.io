Amenazas detectadas. El mainframe está bajo ataque. ¿Tu única arma? **Tu teclado.**

Escribe las palabras que caen antes de que lleguen al mainframe. Encadena combos para puntos extra. ¿Cuánto tiempo puedes defender el sistema?

## Cómo Jugar

- **Escribe** las palabras que caen para destruirlas antes de que toquen el mainframe.
- **El matching** comienza automáticamente — escribe la primera letra de cualquier amenaza.
- **Los combos** multiplican tu puntuación — encadena eliminaciones sin fallar.
- **La velocidad aumenta** con cada nivel. Mantente alerta.
- **Toca** la pantalla en móvil para abrir el teclado.
