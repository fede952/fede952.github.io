Bedrohungen erkannt. Der Mainframe wird angegriffen. Deine einzige Waffe? **Deine Tastatur.**

Tippe die fallenden Wörter, bevor sie den Mainframe erreichen. Baue Combos für Bonuspunkte auf. Wie lange kannst du das System verteidigen?

## So wird gespielt

- **Tippe** die fallenden Wörter, um sie zu zerstören, bevor sie den Mainframe treffen.
- **Das Matching** startet automatisch — tippe den ersten Buchstaben einer Bedrohung.
- **Combos** multiplizieren deinen Score — verkette Abschüsse ohne Fehler.
- **Die Geschwindigkeit steigt** mit jedem Level. Bleib wachsam.
- **Tippe** auf den Bildschirm am Handy, um die Tastatur zu öffnen.
