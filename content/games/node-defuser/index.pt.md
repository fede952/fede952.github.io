A rede está infectada com **15 nós de vírus ocultos**. Use a lógica para revelar células seguras sem ativar malware. Os números indicam quantos vírus cercam cada célula.

{{< game-minesweeper >}}

## Como Jogar

- **Clique esquerdo** para revelar uma célula
- **Clique direito** (desktop) ou **pressão longa** (celular) para marcar um vírus suspeito
- Os números mostram quantas células adjacentes contêm vírus
- Revele todas as células seguras para desarmar a grade
