La red está infectada con **15 nodos de virus ocultos**. Usa la lógica para revelar celdas seguras sin activar malware. Los números indican cuántos virus rodean cada celda.

{{< game-minesweeper >}}

## Cómo Jugar

- **Clic izquierdo** para revelar una celda
- **Clic derecho** (escritorio) o **pulsación larga** (móvil) para marcar un virus sospechoso
- Los números muestran cuántas celdas adyacentes contienen virus
- Revela todas las celdas seguras para desactivar la cuadrícula
