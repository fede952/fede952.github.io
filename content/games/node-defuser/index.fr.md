Le réseau est infecté par **15 nœuds virus cachés**. Utilisez la logique pour révéler les cellules sûres sans déclencher de malware. Les chiffres indiquent combien de virus entourent chaque cellule.

{{< game-minesweeper >}}

## Comment Jouer

- **Clic gauche** pour révéler une cellule
- **Clic droit** (bureau) ou **appui long** (mobile) pour marquer un virus suspect
- Les chiffres montrent combien de cellules adjacentes contiennent des virus
- Révélez toutes les cellules sûres pour désamorcer la grille
