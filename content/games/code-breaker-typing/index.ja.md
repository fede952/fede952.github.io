ターゲットシステムは暗号化されたファイアウォールの向こうにロックされている。侵入する唯一の方法は？**制限時間内に正確なコードシーケンスを入力すること。** 一つでもミスすればロックアウトされる。君の指はどれだけ速い？

{{< code-breaker >}}

## 遊び方

- **「INITIALIZE HACK」をクリック**してタイマーを開始し、コードスニペットを表示させる。
- **表示されたスニペットを正確に入力**する — すべての文字、スペース、記号が重要だ。
- **30秒以内にクリア**してシステムに侵入せよ。
- 時間切れになると接続が切断される。**RETRY HACK** を押して再挑戦しよう。
