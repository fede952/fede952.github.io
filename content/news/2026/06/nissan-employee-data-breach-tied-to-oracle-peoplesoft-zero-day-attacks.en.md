Nissan warns of a data breach affecting current and former employees after threat actors exploited an Oracle PeopleSoft vulnerability, linked to ShinyHunters.

{{< cyber-report severity="High" source="BleepingComputer" target="Oracle PeopleSoft servers" >}}

Nissan has disclosed a data breach impacting current and former employees after threat actors exploited a vulnerability in Oracle PeopleSoft. The attacks are linked to the ShinyHunters extortion group, which previously targeted similar systems.

{{< ad-banner >}}

The breach involved unauthorized access to employee data, though specific details on the vulnerability exploited have not been disclosed. Nissan is notifying affected individuals and advising them to monitor for identity theft.

This incident highlights the ongoing risk of zero-day vulnerabilities in enterprise resource planning (ERP) systems like Oracle PeopleSoft. Organizations using such platforms should prioritize patching and monitor for unusual activity.

{{< netrunner-insight >}}

For SOC analysts, this breach underscores the need to monitor Oracle PeopleSoft logs for anomalous access patterns, especially from unknown IPs. DevSecOps teams should ensure that PeopleSoft instances are patched promptly and that web application firewalls are configured to block common exploitation attempts.
