Elastic Security Labs раскрывает кампанию, использующую вредоносные Google Ads для распространения загрузчика OXLOADER, который доставляет вредоносное ПО CastleStealer, вероятно, управляемое русскоязычными злоумышленниками.

{{< cyber-report severity="High" source="The Hacker News" target="Пользователи, кликающие по вредоносным Google Ads" >}}

Исследователи в области кибербезопасности из Elastic Security Labs обнаружили новую кампанию, использующую вредоносные Google Ads для распространения ранее неизвестного загрузчика вредоносного ПО под названием OXLOADER. Загрузчик используется для доставки CastleStealer — вредоносной программы, крадущей учетные данные, ничего не подозревающим жертвам.

{{< ad-banner >}}

Считается, что кампания имеет финансовую мотивацию и, вероятно, управляется русскоязычными злоумышленниками. Использование Google Ads в качестве начального вектора заражения подчеркивает эволюцию тактик киберпреступников, направленных на обход традиционных мер безопасности и охват более широкой аудитории.

Организациям и частным лицам рекомендуется проявлять осторожность при клике на рекламу, даже если она кажется исходящей из легитимных источников. Использование блокировщиков рекламы и поддержание актуального антивирусного программного обеспечения может помочь снизить риск таких атак.

Для аналитиков SOC критически важно отслеживать необычные клики по рекламе и последующие сетевые подключения к неизвестным доменам. Командам DevSecOps следует рассмотреть возможность блокировки доменов, связанных с рекламой, в прокси-фильтрах и информировать пользователей о рисках клика по рекламе, даже в доверенных поисковых системах.
