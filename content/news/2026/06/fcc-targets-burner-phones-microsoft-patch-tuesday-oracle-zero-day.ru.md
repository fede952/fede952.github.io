FCC предлагает правила для прекращения использования анонимных одноразовых телефонов. Крупнейший вторник исправлений Microsoft включает охоту за ошибками с помощью ИИ. ShinyHunters эксплуатирует уязвимость Oracle zero-day.

{{< cyber-report severity="High" source="Wired Security" target="Системы Oracle" >}}

FCC предложила новые правила, направленные на прекращение использования одноразовых телефонов, которые часто используются для незаконной деятельности из-за своей анонимности. Этот шаг является частью более широких усилий по повышению безопасности и отслеживаемости в коммуникациях.

{{< ad-banner >}}

Другие новости: Microsoft выпустила крупнейшее обновление во вторник исправлений, основанное на обнаружении уязвимостей с помощью ИИ. Обновление устраняет рекордное количество ошибок, подчеркивая растущую роль искусственного интеллекта в кибербезопасности.

Сообщается, что группа вымогателей ShinyHunters эксплуатирует уязвимость Oracle zero-day. Подробности атаки все еще выясняются, но организациям, использующим продукты Oracle, настоятельно рекомендуется немедленно установить исправления.

{{< netrunner-insight >}}

Аналитикам SOC следует уделить первоочередное внимание мониторингу эксплуатации уязвимости Oracle zero-day, так как ShinyHunters известна быстрой разработкой эксплойтов. Командам DevSecOps следует оценить инструменты управления исправлениями на основе ИИ, чтобы успевать за расширяющимся циклом обновлений Microsoft. Предложение FCC по одноразовым телефонам может сигнализировать о будущих нормативных изменениях, влияющих на практики операционной безопасности.
