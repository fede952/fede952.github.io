Bundesbehörden müssen bis Sonntag eine Schwachstelle in Cisco Unified Communications Manager patchen, da CISA vor aktiver Ausnutzung warnt.

{{< cyber-report severity="High" source="BleepingComputer" target="Cisco Unified Communications Manager Server" >}}

Die US-amerikanische Cybersecurity and Infrastructure Security Agency (CISA) hat eine dringende Anweisung erlassen, die Bundesbehörden verpflichtet, bis Sonntag eine Schwachstelle im Cisco Unified Communications Manager Server zu patchen. Der Fehler wird Berichten zufolge aktiv in Angriffen ausgenutzt, obwohl in den verfügbaren Informationen keine spezifischen CVE-Identifikatoren oder technischen Details offengelegt wurden.

{{< ad-banner >}}

Cisco Unified Communications Manager ist eine kritische Komponente für Unternehmenssprach- und Videokommunikation und stellt daher ein hochwertiges Ziel für Bedrohungsakteure dar. Die kurze Behebungsfrist unterstreicht den Schweregrad der Bedrohung und die Notwendigkeit einer schnellen Patchung auf betroffenen Systemen.

Auch Organisationen außerhalb der Bundesregierung wird dringend empfohlen, diesen Patch zu priorisieren. Angesichts der aktiven Ausnutzung könnten Verzögerungen bei der Behebung zu Netzwerkkompromittierung, Datendiebstahl oder weiterer lateraler Bewegung in betroffenen Umgebungen führen.
