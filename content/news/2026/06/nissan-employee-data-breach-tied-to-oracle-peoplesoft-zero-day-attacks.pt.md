A Nissan alerta sobre uma violação de dados que afeta funcionários atuais e antigos, após agentes de ameaças explorarem uma vulnerabilidade no Oracle PeopleSoft, ligada ao ShinyHunters.

{{< cyber-report severity="High" source="BleepingComputer" target="Servidores Oracle PeopleSoft" >}}

A Nissan divulgou uma violação de dados que impacta funcionários atuais e antigos, após agentes de ameaças explorarem uma vulnerabilidade no Oracle PeopleSoft. Os ataques estão ligados ao grupo de extorsão ShinyHunters, que já alvejou sistemas semelhantes anteriormente.

{{< ad-banner >}}

A violação envolveu acesso não autorizado a dados de funcionários, embora detalhes específicos sobre a vulnerabilidade explorada não tenham sido divulgados. A Nissan está notificando os indivíduos afetados e aconselhando-os a monitorar sinais de roubo de identidade.

Este incidente destaca o risco contínuo de vulnerabilidades de dia zero em sistemas de planejamento de recursos empresariais (ERP) como o Oracle PeopleSoft. Organizações que utilizam tais plataformas devem priorizar a aplicação de patches e monitorar atividades suspeitas.

{{< netrunner-insight >}}

Para analistas de SOC, esta violação ressalta a necessidade de monitorar logs do Oracle PeopleSoft em busca de padrões de acesso anômalos, especialmente de IPs desconhecidos. As equipes de DevSecOps devem garantir que as instâncias do PeopleSoft sejam atualizadas prontamente e que firewalls de aplicação web estejam configurados para bloquear tentativas comuns de exploração.
