Microsoft предупреждает о активной фишинговой кампании, нацеленной на отели в Европе и Азии, с использованием ZIP-файлов с фото-тематикой, которые загружают Node.js имплант.

{{< cyber-report severity="High" source="The Hacker News" target="гостиничные и гостинично-ресторанные организации" >}}

С апреля 2026 года активная фишинговая кампания нацелена на гостиничные и гостинично-ресторанные организации по всей Европе и Азии. Злоумышленники используют ZIP-файлы с фото-тематикой в качестве приманок, которые при выполнении загружают Node.js имплант на компьютеры стойки регистрации.

{{< ad-banner >}}

Microsoft не связывает эту активность ни с одной известной угрозой, и конечная цель операторов остается неясной. Приманка специально разработана для эксплуатации особенностей работы отелей, что указывает на целенаправленный подход социальной инженерии.

Node.js имплант предоставляет злоумышленникам точку опоры в целевых сетях, потенциально позволяя боковое перемещение и кражу данных. Организациям в гостиничном секторе рекомендуется проявлять осторожность с нежелательными вложениями электронной почты и отслеживать подозрительные процессы Node.js.

{{< netrunner-insight >}}

Аналитикам SOC следует отслеживать необычные процессы Node.js и исходящие соединения с систем стойки регистрации. Командам DevSecOps следует рассмотреть возможность блокировки выполнения скриптов Node.js из вложений электронной почты и внедрение белых списков приложений для смягчения таких имплантов.
