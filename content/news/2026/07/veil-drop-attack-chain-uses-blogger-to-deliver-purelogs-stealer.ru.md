Исследователи обнаружили многоэтапную кампанию по распространению вредоносного ПО, использующую страницы Blogger и методы социальной инженерии для доставки стилера информации PureLogs, получившую название VEIL#DROP.

{{< cyber-report severity="High" source="The Hacker News" target="Пользователи платформы Blogger" >}}

Специалисты по кибербезопасности выявили новую многоэтапную цепочку доставки вредоносного ПО, названную VEIL#DROP компанией Securonix, которая использует социальную инженерию и страницы Blogger для распространения стилера информации PureLogs. Предполагается, что начальные полезные нагрузки доставляются через целевые фишинговые атаки или компрометацию при посещении веб-сайтов, когда ничего не подозревающие пользователи переходят на вредоносные страницы Blogger.

{{< ad-banner >}}

Цепочка атак включает несколько этапов, при этом платформа Blogger служит механизмом хостинга вредоносного контента. После того как пользователь посещает скомпрометированную страницу, вредоносное ПО загружается и выполняется, что приводит к краже конфиденциальной информации. PureLogs — известный стилер, нацеленный на учетные данные, данные браузера и другую личную информацию.

Эта кампания подчеркивает растущее использование легитимных платформ, таких как Blogger, для хостинга вредоносных нагрузок, что усложняет обнаружение. Организациям следует информировать пользователей о рисках перехода по ненадежным ссылкам и внедрять надежные средства фильтрации электронной почты и веб-трафика для снижения подобных угроз.

{{< netrunner-insight >}}

Для аналитиков SOC: отслеживайте необычные исходящие соединения с доменами Blogger и проверяйте трафик на наличие закодированных полезных нагрузок. Командам DevSecOps следует внедрить строгий список разрешенных веб-сервисов и развернуть правила обнаружения на конечных точках для индикаторов PureLogs. Использование легитимных платформ для хостинга вредоносного ПО подчеркивает необходимость обнаружения на основе поведения, а не простого блокирования доменов.
