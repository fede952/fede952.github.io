A gigante de cosméticos Estée Lauder notifica clientes sobre uma violação de dados após hackers explorarem uma vulnerabilidade no Oracle E-Business Suite usado para operações de RH.

{{< cyber-report severity="High" source="BleepingComputer" target="Sistemas de RH do Oracle E-Business Suite" >}}

A Estée Lauder, empresa global de cosméticos, está notificando clientes sobre uma violação de dados que ocorreu após atacantes explorarem uma vulnerabilidade no Oracle E-Business Suite. O software era usado pela empresa para operações de recursos humanos, sugerindo que a violação pode ter exposto dados sensíveis de funcionários ou clientes.

{{< ad-banner >}}

O incidente destaca os riscos associados a vulnerabilidades em softwares empresariais de terceiros, especialmente aqueles usados para funções críticas de negócios como RH. Embora detalhes técnicos específicos da exploração não tenham sido divulgados, organizações que usam Oracle E-Business Suite devem priorizar a aplicação de patches e o monitoramento de atividades suspeitas.

A Estée Lauder ainda não divulgou um cronograma completo ou a extensão da violação, mas os clientes afetados estão sendo notificados. Este incidente serve como lembrete de que mesmo grandes empresas com programas de segurança maduros podem ser vítimas de vulnerabilidades não corrigidas em aplicativos de negócios amplamente utilizados.

{{< netrunner-insight >}}

Esta violação ressalta a importância de uma gestão rigorosa de patches para aplicativos empresariais como Oracle E-Business Suite. Analistas de SOC devem monitorar tentativas de exploração contra esta plataforma, enquanto equipes DevSecOps devem garantir a implantação oportuna de atualizações de segurança e implementar controles compensatórios como segmentação de rede e regras de WAF.
