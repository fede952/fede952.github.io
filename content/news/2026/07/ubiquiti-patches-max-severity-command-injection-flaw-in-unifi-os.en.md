Ubiquiti released security updates for seven critical vulnerabilities in UniFi OS, including a maximum-severity command injection flaw. Admins urged to patch immediately.

{{< cyber-report severity="Critical" source="BleepingComputer" target="UniFi OS" >}}

Ubiquiti has released security updates to address seven critical vulnerabilities in its UniFi OS, the operating system powering its networking devices. The most severe of these is a maximum-severity flaw that can be exploited in command injection attacks, potentially allowing attackers to execute arbitrary commands on affected systems.

{{< ad-banner >}}

The vulnerabilities affect a wide range of Ubiquiti products running UniFi OS. While specific CVE identifiers were not disclosed in the initial advisory, the company has urged all users to apply the latest firmware updates immediately to mitigate the risk of exploitation.

Given the critical nature of these vulnerabilities, particularly the command injection flaw, organizations using Ubiquiti equipment should prioritize patching. No evidence of active exploitation has been reported at this time, but the severity warrants immediate attention from security teams.
