Ubiquiti выпустила обновления безопасности для семи критических уязвимостей в UniFi OS, включая ошибку внедрения команд максимальной степени серьезности. Администраторам настоятельно рекомендуется немедленно установить исправления.

{{< cyber-report severity="Critical" source="BleepingComputer" target="UniFi OS" >}}

Компания Ubiquiti выпустила обновления безопасности для устранения семи критических уязвимостей в своей операционной системе UniFi OS, которая используется в сетевых устройствах. Наиболее серьезной из них является ошибка максимальной степени серьезности, которая может быть использована в атаках с внедрением команд, потенциально позволяя злоумышленникам выполнять произвольные команды на затронутых системах.

{{< ad-banner >}}

Уязвимости затрагивают широкий спектр продуктов Ubiquiti, работающих под управлением UniFi OS. Хотя в первоначальном уведомлении не были указаны конкретные идентификаторы CVE, компания призвала всех пользователей немедленно установить последние обновления прошивки, чтобы снизить риск эксплуатации.

Учитывая критический характер этих уязвимостей, особенно ошибки внедрения команд, организациям, использующим оборудование Ubiquiti, следует уделить первостепенное внимание установке исправлений. На данный момент нет сообщений об активной эксплуатации, но серьезность требует немедленного внимания со стороны команд безопасности.

Это ситуация обязательного исправления для любой организации, использующей оборудование Ubiquiti UniFi. Ошибки внедрения команд в сетевой инфраструктуре — мечта любой команды красных, поэтому ожидайте, что эксплойты с доказательством концепции появятся быстро. Уделите первостепенное внимание установке исправлений и отслеживайте аномальные исходящие соединения с затронутых устройств.
