Группа угроз RansomHouse взяла на себя ответственность за взлом репозитория исходного кода Trellix, опубликовав изображения в качестве доказательства проникновения.

{{< cyber-report severity="High" source="BleepingComputer" target="репозиторий исходного кода Trellix" >}}

Группа угроз RansomHouse взяла на себя ответственность за взлом репозитория исходного кода Trellix, о котором было объявлено на прошлой неделе. В качестве доказательства проникновения группа опубликовала небольшой набор изображений, указывающих на доступ к конфиденциальному коду.

{{< ad-banner >}}

Атака подчеркивает сохраняющиеся риски для цепочек поставок программного обеспечения, где репозитории исходного кода являются главными целями злоумышленников. Хотя полный масштаб взлома пока неизвестен, инцидент подчеркивает необходимость надежного контроля доступа и мониторинга сред разработки.

Trellix еще не опубликовала подробного заявления о последствиях взлома. Специалистам по безопасности рекомендуется пересмотреть практики безопасности собственных репозиториев кода и обеспечить внедрение многофакторной аутентификации и доступа с минимальными привилегиями.

Для аналитиков SOC: отслеживайте необычные паттерны доступа к репозиториям исходного кода и рассмотрите возможность развертывания honeytokens. Командам DevSecOps следует внедрять строгие правила защиты веток и аудит всех сторонних интеграций с платформами хостинга кода.
