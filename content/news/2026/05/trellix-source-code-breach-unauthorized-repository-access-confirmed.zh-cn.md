Trellix确认其部分源代码仓库遭到未经授权访问，正与取证专家和执法部门合作调查此次泄露事件。

{{< cyber-report severity="High" source="The Hacker News" target="Trellix源代码仓库" >}}

网络安全公司Trellix披露了一起安全事件，涉及对其部分源代码仓库的未经授权访问。该公司表示，其“近期发现”了这一入侵行为，并立即聘请了领先的取证专家进行调查和修复。

{{< ad-banner >}}

Trellix还已通知执法部门此次泄露，但尚未披露暴露代码的范围或是否有客户数据受到影响。公告中未包含关于攻击向量或入侵时间范围的具体技术细节。

作为网络安全行业的主要参与者，Trellix的源代码泄露引发了对潜在知识产权盗窃以及威胁行为者分析代码以寻找漏洞的可能性的担忧。该公司目前未表示其产品或服务受到任何影响。
