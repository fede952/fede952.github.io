RansomHouse 위협 그룹이 Trellix 소스 코드 저장소 침해에 대한 책임을 주장하며 침입 증거로 이미지를 유출했습니다.

{{< cyber-report severity="High" source="BleepingComputer" target="Trellix 소스 코드 저장소" >}}

RansomHouse 위협 그룹이 지난주 공개된 Trellix의 소스 코드 저장소 침해에 대한 책임을 주장했습니다. 침입 증거로 소규모 이미지 세트를 유출하여 민감한 코드에 접근했음을 나타냈습니다.

{{< ad-banner >}}

이번 공격은 소스 코드 저장소가 위협 행위자의 주요 표적이 되는 소프트웨어 공급망의 지속적인 위험을 강조합니다. 침해의 전체 범위는 아직 알려지지 않았지만, 이 사건은 개발 환경에 대한 강력한 접근 통제 및 모니터링의 필요성을 강조합니다.

Trellix는 아직 침해 영향에 대한 상세한 성명을 발표하지 않았습니다. 보안 팀은 자체 코드 저장소 보안 관행을 검토하고 다중 인증 및 최소 권한 접근이 적용되도록 해야 합니다.
