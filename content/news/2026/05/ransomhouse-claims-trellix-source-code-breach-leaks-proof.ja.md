RansomHouse脅威グループがTrellixのソースコードリポジトリ侵害の責任を主張し、侵入の証拠として画像をリークしました。

{{< cyber-report severity="High" source="BleepingComputer" target="Trellixソースコードリポジトリ" >}}

RansomHouse脅威グループは、先週公開されたTrellixのソースコードリポジトリ侵害の責任を主張しました。侵入の証拠として、グループは少数の画像をリークし、機密コードにアクセスしていたことを示しました。

{{< ad-banner >}}

この攻撃は、ソフトウェアサプライチェーンに対する継続的なリスクを浮き彫りにしており、ソースコードリポジトリは脅威アクターの主要な標的です。侵害の全容はまだ明らかになっていませんが、このインシデントは、開発環境に対する堅牢なアクセス制御と監視の必要性を強調しています。

Trellixは侵害の影響に関する詳細な声明をまだ発表していません。セキュリティチームは、自社のコードリポジトリのセキュリティ慣行を見直し、多要素認証と最小権限アクセスが適用されていることを確認することを推奨します。
