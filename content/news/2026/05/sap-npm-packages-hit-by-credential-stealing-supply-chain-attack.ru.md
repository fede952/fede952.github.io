Кампания под названием 'Mini Shai-Hulud' нацелена на npm-пакеты, связанные с SAP, с помощью вредоносного ПО, крадущего учетные данные, затрагивая несколько пакетов. Исследователи из нескольких фирм предупреждают о рисках для цепочки поставок.

{{< cyber-report severity="High" source="The Hacker News" target="npm-пакеты, связанные с SAP" >}}

Исследователи в области кибербезопасности обнаружили кампанию атаки на цепочку поставок, нацеленную на npm-пакеты, связанные с SAP. Названная 'Mini Shai-Hulud', кампания развертывает вредоносное ПО для кражи учетных данных через скомпрометированные пакеты, согласно отчетам Aikido Security, Onapsis, OX Security, SafeDep, Socket, StepSecurity и Wiz.

{{< ad-banner >}}

Атака затрагивает несколько npm-пакетов, связанных с SAP, хотя конкретные названия пакетов и версии не раскрыты. Вредоносное ПО предназначено для кражи учетных данных, потенциально предоставляя злоумышленникам доступ к чувствительным средам SAP и нижестоящим системам.

Этот инцидент подчеркивает растущую угрозу для цепочек поставок программного обеспечения, особенно для критически важных для предприятия платформ, таких как SAP. Организациям, использующим затронутые пакеты, рекомендуется провести аудит своих зависимостей и сменить все потенциально скомпрометированные учетные данные.

{{< netrunner-insight >}}

Для аналитиков SOC и команд DevSecOps эта атака подчеркивает необходимость тщательного сканирования зависимостей и проверки целостности npm-пакетов. Отслеживайте необычные исходящие соединения из систем, связанных с SAP, и рассмотрите возможность внедрения самозащиты приложений во время выполнения (RASP) для обнаружения кражи учетных данных. Немедленно смените все учетные данные, которые могли быть раскрыты через скомпрометированные пакеты.
