CISAは、ゼロデイ攻撃で悪用された高深刻度のIvanti Endpoint Manager Mobileの脆弱性に対する緊急パッチを義務付け、連邦政府機関に4日以内のネットワーク保護を求める。

{{< cyber-report severity="High" source="BleepingComputer" target="Ivanti Endpoint Manager Mobile (EPMM)" >}}

米国サイバーセキュリティ・インフラストラクチャセキュリティ庁（CISA）は、Ivanti Endpoint Manager Mobile（EPMM）の高深刻度の脆弱性を4日以内にパッチするよう連邦政府機関に義務付ける拘束力のある運用指令を発出した。この欠陥はゼロデイ攻撃で活発に悪用されており、さらなる侵害を防ぐための緊急行動が促されている。

{{< ad-banner >}}

最初の報告では特定のCVE識別子やCVSSスコアは開示されなかったものの、この指令は脆弱性の重大性を強調している。Ivanti EPMMはモバイルデバイス管理に広く使用されており、脅威アクターがエンタープライズネットワークへの初期アクセスを得るための価値の高い標的となっている。

連邦政府以外の組織も、パッチ適用を優先し、Ivanti EPMMの導入環境に侵害の兆候がないか確認することを強く推奨する。CISAの短い是正期間は、脅威の深刻さと広範囲にわたる悪用の可能性を反映している。

{{< netrunner-insight >}}

SOCアナリストは、Ivanti EPMMに関連する侵害指標を直ちに確認し、4日間の猶予期間内にパッチ適用を優先すべきである。DevSecOpsチームは、カスタム統合や構成がセキュリティ更新の展開を遅らせないようにする必要がある。この脆弱性はすでに活発に悪用されているためである。
