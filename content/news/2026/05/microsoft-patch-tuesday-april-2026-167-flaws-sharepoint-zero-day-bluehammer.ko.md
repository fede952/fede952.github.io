마이크로소프트가 셰어포인트 제로데이와 공개적으로 알려진 Windows Defender 결함(블루해머)을 포함한 167개의 취약점을 수정했습니다. 구글 크롬과 어도비 리더도 활발히 악용되는 버그를 패치했습니다.

{{< cyber-report severity="Critical" source="Krebs on Security" target="Microsoft Windows, SharePoint, Windows Defender, Chrome, Adobe Reader" >}}

마이크로소프트의 2026년 4월 패치 화요일은 Windows 및 관련 소프트웨어 전반에 걸쳐 무려 167개의 보안 취약점을 해결합니다. 가장 중요한 것 중 하나는 원격 코드 실행을 허용할 수 있는 셰어포인트 서버 제로데이 취약점이지만, 보고서에는 CVE 식별자가 제공되지 않았습니다. 또한 '블루해머'라고 불리는 Windows Defender의 공개적으로 알려진 약점이 수정되었습니다.

{{< ad-banner >}}

별도로, 구글 크롬은 2026년 네 번째 제로데이를 패치하여 빈번한 브라우저 업데이트 추세를 이어갔습니다. 어도비 리더도 원격 코드 실행으로 이어질 수 있는 활발히 악용되는 결함을 해결하기 위해 긴급 업데이트를 받았습니다. 조직은 활발한 악용을 고려하여 이러한 업데이트를 우선시해야 합니다.

이번 달의 방대한 패치 양은 강력한 패치 관리 프로세스의 중요성을 강조합니다. 보안 팀은 셰어포인트 제로데이와 Windows Defender 문제를 즉시 우선순위로 삼아야 하며, 동시에 엔터프라이즈 전반에 걸쳐 크롬과 어도비 리더가 업데이트되도록 해야 합니다.

SOC 분석가의 경우, 셰어포인트 제로데이와 블루해머 Windows Defender 결함을 즉시 패치하여 우선순위를 두십시오. 이들은 활발히 악용되거나 공개적으로 알려져 있습니다. DevSecOps 팀은 이러한 업데이트를 CI/CD 파이프라인에 통합하고 엔드포인트 보호 도구가 Defender 수정으로 인해 중단되지 않는지 확인해야 합니다. 크롬과 어도비 리더 패치도 활발한 악용 상태를 고려하여 긴급한 주의가 필요합니다.
